✦ Завдання (Частина 2 — CSS)
▶

В <style> задати стилі:

  1. body:
    • Фон — зображення mario_background.jpg
    • Фон має покривати весь екран: background-size: cover;
    • Основний шрифт — Comic Sans MS.
    • text-align: center;

  2. h1:
    • Колір — red.
    • Розмір шрифту — 38px.

  3. h2:
    • Колір — darkblue.
    • Розмір шрифту — 28px.

  4. p:
    • Розмір шрифту — 30px.

  5. img:
    • Висота — 300px.
    • Рамка — 5px solid brown.
    • Закруглення рамки — 10px.

  6. #menu:
    • Фон — white.

  7. a:
    • Колір тексту — red.
    • Фон — white.
    • Розмір шрифту — 20px.
    • Без підкреслення — text-decoration: none;
    • margin: 10px;.

✧ Підказки: CSS
▶

Картинка як фон сторінки

body
{
  background-image: url(""); /* поставити картинку на фон */
  background-repeat: no-repeat; /* прибрати повторення */
  background-size: cover; /* розтягнути на весь екран */
}

Розташувати всі елементи по центру

body
{
  text-align: center;
}

Шрифт

font-family: "Comic Sans MS"; /* сімейство шрифтів */
font-size: 25px; /* розмір шрифту */

Колір тексту та фону

color: red; /* колір тексту */
background-color: green; /* колір фону */

Рамка

border: 2px solid red; /* ширина стиль колір */
border-radius: 10px; /* заокруглення кутів */

/* можна також задати властивості рамки окремо */
border-width: 2px; /* ширина */
border-style: solid; /* стиль */
border-color: red; /* колір */

Стилі рамок:

Ширина і висота

width: 200px; /* ширина */
height: 100px; /* висота */

Стилізація посилань

a 
{
  text-decoration: none; /* прибрати підкреслення тексту */
}

/* стилі при наведенні курсора мишки */
a:hover 
{
  color: red; 
}
