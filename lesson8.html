Гіперосилання

✦ Завдання
▶
  1. Завантажте файли для виконання завдання:
    • index8_1.html
    • mario.html
  2. Завантажте картинку для фону сторінки: background.jpg.
  3. Перемістіть всі файли в свою папку. Картинка має бути в папці images.
  4. Виконайте вказівки, описані в коментарях в index8_1.html та mario.html.

✧ Підказки: HTML
▶

Посилання

<a href="адреса_посилання">Текст посилання</a>

Відкрити посилання в новій вкладці

<a href="адреса_посилання" target="_blank">Текст посилання</a>
  • target="_self" відкрити в тій самій вкладці (по-замовчуванню)
  • target="_blank" — відкрити в новій вкладці

Посилання на зовнішній сайт

<a href="адреса_сайту">Текст посилання</a>

<a href="https://www.wikipedia.org/">Wikipedia</a>

Посилання на .html файл

<a href="назва_файлу">Текст посилання</a>
              
<a href="page2.html">Page 2</a>

Посилання на елементи сторінки

<a href="#mario">Посилання на Маріо</a>

...

<h2 id="mario">Маріо</h2>
  • id — атрибут, який дозволяє дати елементу нове ім'я.
  • Кожен id на сторінці має бути унікальний (не можна давати однакове значення двом різним елементам).
✧ Підказки: CSS
▶

Стилізація посилань

a 
{
  text-decoration: none; /* прибрати підкреслення тексту */
  color: yellow; 
}

/* стилі при наведенні курсора мишки */
a:hover 
{
  text-decoration: underline; /* додати підкреслення тексту */
  color: green; 
}
  • За замовчуванням посилання мають підкреслення тексту і синій колір.
