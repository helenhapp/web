Тема 9

Посилання та картинки

✦ Завдання
▶
  1. Завантажте файл для виконання завдання: index9_2.html.
  2. Завантажте картинку для фону сторінки: Pokemons.png.
  3. Перемістіть обидва файли в свою папку. Картинка має бути в папці images.
  4. Виконайте вказівки, описані в коментарях в index9_2.html.

✧ Підказки: HTML
▶

Структура сторінки

<html>
  <head>
    <style></style>
  </head>
  <body></body>
</html>

Заголовки

<h1>Заголовок 1-го рівня</h1>

Параграфи

<p>Параграф тексту</p>

Посилання

<a href="адреса_посилання" target="_blank">Текст посилання</a>
  • target="_self" відкрити в тій самій вкладці (по-замовчуванню)
  • target="_blank" — відкрити в новій вкладці

Картинки

<img src="джерело_картинки" alt="альтернативний_текст">

Посилання + Картинки

<a href="" target="">
  <img src="" alt="">
</a>

<!-- Посилання на зовнішній сайт -->
<a href="https://en.wikipedia.org/wiki/Pikachu" target="_blank">
  <img src="images/Pokemons.png" alt="покемони">
</a>

<!-- Відкрити картинку в новій вкладці -->
<a href="images/Pokemons.png" target="_blank">
  <img src="images/Pokemons.png" alt="покемони">
</a>

<!-- Завантажити картинку при натисканні -->
<a href="images/Pokemons.png" download>
  <img src="images/Pokemons.png" alt="покемони">
</a>
✧ Підказки: CSS
▶

Картинка як фон сторінки

body
{
  background-image: url(""); /* поставити картинку на фон */
  background-repeat: no-repeat; /* прибрати повторення */
  background-size: cover; /* розтягнути на весь екран */
  background-position: center; /* розташувати по центру */
  min-height: 100vh; /* мінімальна висона сторінки: висота екрану */
}

Розташувати всі елементи по центру

body
{
  text-align: center;
}

Шрифт

font-family: "Comic Sans MS"; /* сімейство шрифтів */
font-size: 25px; /* розмір шрифту */

Колір тексту та фону

color: red; /* колір тексту */
background-color: green; /* колір фону */

Рамка

border: 2px solid red; /* ширина стиль колір */
border-radius: 10px; /* заокруглення кутів */

/* можна також задати властивості рамки окремо */
border-width: 2px; /* ширина */
border-style: solid; /* стиль */
border-color: red; /* колір */

Ширина і висота

width: 200px; /* ширина */
height: 100px; /* висота */

Стилізація посилань

a 
{
  text-decoration: none; /* прибрати підкреслення тексту */
}

/* стилі при наведенні курсора мишки */
a:hover 
{
  color: red; 
}

/* стилі при виборі посилання */
a:focus 
{
  color: green; 
}

/* стилі при натисканні на посилання */
a:active 
{
  color: blue; 
}
